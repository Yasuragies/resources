In this page we're going to learn how to tell time in Japanese.
But first let us meet our new friend : に partice.

There are many way to use に particle but as for now let's focus on how to use に particle to tell time.
we usually attach particle に after the time.

Example :

３時じ　に、ラーメン を 食たべます。
(I will) eat ramen at 3 A.m

You can also distinguist between P.m and A.m by using these 2 kanji
午前ごぜん = A.m
午後ごご = P.m

usually we put 午前ごぜん and 午後ごご before the Kanji.

Example :

午前ごぜん９時くじ = 9 A.m
午後ごご３時さんじ = 3 P.m

史子ふみこは、午後ごご４時よじ に 図書館としょかん で 本ほん を 読よ 読みました。
Fumiko read a book at the library at 4 P.m

but に particle cannot be used for relative time such as :

昨日きのう = yesterday
今日きょう = today
明日あした = tomorrow
先週せんしゅう = last week
今週こんしゅう = this week
来週らいしゅう = next week
先月せんげつ = last month
今月こんげつ = this month
来月らいげつ = next month
去年きょねん = last year
今年ことし = this year
来年らいねん = next year

Example :

私 は 昨日きのう、学校がっこう に 行いきました。
I went to school yesterday.

But as we know it, Japanese doesn't use the word 私 too much.
Then if the topic is not present, the relative time particle will take the topic place

昨日きのう は 学校がっこう に 行いきました。
I went to school yesterday.

So following basic Japanese sentence structure :

Topic + Time (day) + time (clock) + location + verb

More example :

立花たちばな は 昨日きのう、午前ごぜん ３時さんじ に、部屋へや で 泣なきました。
Yesterday, Tachibana cried in (his) room at 3 A.m (relatable eh..).

ゆりこ は 明日あした、午前ごぜん９時くじ に 東京とうきょう に 行いきました。
