に particle


Vocabulary

行く = いく
寝る = ねる = to sleep
帰る = かえる = to return home
駅 = えき = station
学校 = school
図書館 = library
きっさてん = café
郵便局 = post office
レストラン = restaurant
~時 = じ = O’clock (suffix) e.g. 1 時
午前 = p.m
午後 = a.m
昨日 = きのう = yesterday
今日 = きょう = today
明日 = あした = tomorrow
金曜日 = きんようび = Friday.
来年 = らいねん = next year.
来週 = らいしゅう = next week.
ご飯 = ごはん = meal.
ある　= to exist (object)
いる = to exist (living being)
ここ = here
そこ = there
あそこ = over there (far)

に as destination particle.
A に B
Noun に verb

in this case に mark A (noun) as the destination of our action (B)

駅えき に 行いく
(I) (will) go to the station

きっさてん に　行いく
(I) (will) go to the café


に to mark location
駅えき は ここ に ある
the station is here.

たけし先生せんせい は 学校がっこう に いる
Takeshi sensei is in school.

彼女かのじょ は あそこ に いる
she is over there.

に to mark time
Example

九時くじ に 寝ねる
(I) (will) sleep at 9'clock

ふみこ は 四時よじ に ご飯はん を 食たべる
fumiko (will) eat (her) meal at 4 o'clock

金曜日きんようび は 図書館としょかん に 行いく
(I will) go to library at friday

The "に" particle cannot be used for relative time like

今日 = きょう
来週 = らいしゅう
来年　= らいねん
(day, week, month, year)
thus it is followed by the topic particle "は"

今日きょう は きっさてん に 行いく
today (I) will go to the cafe.

When we set the topic, the time follows right after the topic particle

ふみこ は 明日あした、学校がっこう に　行いく
Fumiko　(will) go to school tommorow

けん は 来週らいしゅう、映画えいが を 見みる
Ken (will) watch a movie next week
