1. これ・それ・あれ・どれ
used to explain an object that we're pointing at
これ : this (when the object is close to the speaker)
それ : that (when the object is close the listener)
あれ : that (when the object is far from bot speaker and listener)
どれ : which

Grammar :
これ・それ・あれ ＋ は + Noun

Example:
これ は ペン
This is a pen

それ は かばん
That is a bag

あれ は かさ
That over there is an umbrella.

A : たけし の 車くるま は どれですか。
A : which one is takeshi's car.

B : それ は たけし の 車くるま です。
B : That is takeshi's car.


2. この・その・あの・どの

この・その・あの・どの is used point at the object directly

Grammar :

この・その・あの・どの + Noun + は

Comparision

これ は 車くるま です 。
This is a car.

この 車くるま は 。。。。
This car is....

Example :

この ぼうし は 黒くろい です
This hat is black

1.
A : たちばな の カバン は どれですか。
A : Which one is Tachibana's bag?

B : これです。
B : This is.

2.
A : この 人 は 誰だれですか。
A : who is this person?

B : この 人 は たちばな です。
B : This person's Tachibana.

3. ここ・そこ・あそこ・どこ

this grammar is used to describe location.

ここ = here
そこ = there
あそこ = over there
どこ = where?

Example :

A : 喫茶店きっさてん はどこですか。
A : Where is the cafe?

B : 喫茶店はあそこです。
B : The cafe is over there.

前まえ = Infrontv 後ろうしろ = behind
となり = right next to
ちかく = close

Grammar :

Noun + の 前まえ = Infront of noun
Noun + の 後ろうしろ = behind of noun
Noun + の となり = right next to noun
Noun + の ちかく = close/nearby to noun


A : 学校がっこう は どこですか。
A : where is the school?

B : 学校がっこう は 郵便局ゆうびんきょく の 前まえです。
B : the school is infront of the station.

A : 本屋ほんや は どこですか。
A : Where is the book store.

B : 本屋ほんや は 病院びょういん の となりです。
B : The book store is right next to the hospital.

A : 駅えき は どこですか。
A : Where is the station?

B : 駅えきは ホテル の ちかくです。
B : The station is nearby/close to the hotel.
