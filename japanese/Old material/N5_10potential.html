Potential conjugation

vocabulary
~ichidan~
 食べる = たべる = to eat
見る = みる = to see/watch
寝る = ねる = to sleep
起きる = おきる = to wake up

~ichidan~
 会う = あう = to meet
待つ = まつ = to wait
帰る = かえる = to return home
読む = よむ = to read
遊ぶ = あそぶ = to play
死ぬ = しぬ = to die
聞く = きく = to listen
行く = いく = to go

irregular verbs
 する = to do
やる = to do

Grammar
Potential form is used if you want to express "I can/can't" as in
"I can't eat that"

Ichidan verb potential form conjugation.
to turn ichidan verb into potential form, replace the る in the end of the verb with られる

食たべる -> 食たべられる
eat -> can eat

見みる ->　見みられる
see -> can see

寝ねる -> 寝ねられる
sleep -> can sleep

起おきる -> 起おきられる
wake up -> can wake up

Godan verb potential form conjugation
conjugate う・つ・る　to potential form.
 to conjugate this category into potential form, replace the　う・つ・る part。
う -> える
つ -> てる
る -> れる

会あう -> 会あえる
meet -> can meet

待まつ -> 待まてる
wait -> can wait

帰かえる -> 帰かえれる
return home -> can return home

conjugate む・ぶ・ぬ　to　to potential form.
 to conjugate this category into potential form, replace the　む・ぶ・ぬ part。
む -> める
ぶ -> べる
ぬ -> ねる

読よむ -> 読よめる
read -> can read

遊あそぶ -> 遊あそべる
play -> can play

死しぬ -> 死しねる
die -> can die

conjugate く　into　to potential form.
 to conjugate く to potential form, replace く with ける

聞きく　-> 聞きける
hear -> can hear

行いく　-> 行いける
go -> can go

conjugate す　into potential form.
 to conjugate す to potential form, replace す with せる

話はなす　-> 話はなせる
speak -> can speak

Irregular verb potential form conjugation
 する -> できる
やる -> やれる

Example:

料理りょうりできる
can cook.

掃除そうじできる
can clean.

勉強べんきょうできる
can study.

turning potential form into negative potential. After we conjugate a verb to potential form replace the る with ない. this rule works for Ichidan, Godan, and Irregular.

Ichidan
食たべられる -> 食たべられない
can eat -> can't eat

見みる -> 見みられない
can see/watch -> can't see/watch

寝ねる -> 寝ねられない
can sleep -> can't sleep

起おきる -> 起おきられない
can wake up -> can't wake up


Godan

会あえる -> 会あえない
can meet -> can't meet

待まてる -> 待まてない
can wait -> can't wait

帰かえれる -> 帰かえれない
can return home -> can't return home

読よめる -> 読よめない
can read -> can't read

遊あそべる -> 遊あそべない
can play -> can't play

死しねる -> 死しねない
can die -> can't die

聞きける -> 聞きけない
can hear -> can't hear

行いける -> 行いけない
can go -> can't go

話はなせる -> 話はなせない
can speak -> can't speak

Irregular

できる -> できない
can do -> can't do

やれる -> やれない
can do -> can't do
