Pointing + の

Required vocabulary

この = this
その ＝ that
あの = that (over there)
車 = くるま = car
さいふ = ひと = wallet
くつ ＝ shoe
人 = ひと = person
時計 = とけい = watch
パソコン = computer
かのじょ = girlfriend (as noun) / she/her (as pronoun)
かれし = boyfriend (noun)
かれ = he/him (pronoun

Grammar
この B
This B

Example

こ の ぺん
This pen

そ の 車くるま
that car.

この 人 は たけし です
This person is Takeshi

あの 人 は たけし の 彼女かのじょ です
That person is Takeshi’s girlfriend

この かばん は たけし の かばん です
That bag is Takeshi’s bag

あの 時計とけい は たけし の 彼女かのじょ の時計とけい です。
That watch is Takehi’s girlfriend watch
