Chapter 1 你好



Intruduction to Mandarin tones



妈 = mā = longer tones = mother
麻 = má = rising tones = plant
马 = mǎ = wave tones = horse
骂 = mà = dropping tones = angry

Becareful not to call your mother horse


Basic vocabulary:



我 = wǒ = i/me
你 = nǐ = you
我们 = wǒmen = we
你们 = nǐmen = you(plural
好 = hǎo = good
对不起 = duìbuqǐ = sorry
没关系 = méi guān xi = it's okay
谢谢 = xièxie = thank you
不客气 = búkèqi = don't worry it

New vocabulary



叫 = jiào = call
什么 = shéme = what
名字 = míngzi = name

New vocabulary



是 = shì = is
老师 = lǎoshī = teacher
吗 = ma = Question particle
学生 = xuésheng = Student
不 = bú = not
不是 = búshì = is not

In class 在教室


Example Sentence



你 是 老师 吗？
nǐ shì lǎoshī ma?
are you a teacher



我 不 是 老师，我 是 学生
wǒ bú shì lǎoshī, wǒ shì xuésheng
im not a teacher, im a student

New vocabulary



人 = rén = Person
中国 = zhōng guó = China
美国 = měi guó = America
中国人 = zhōng guó rén = Chinese (Person)
美国人 = měi guó rén = American (Person)



你 是 中国人 吗？
nǐ shì zhōng guó rén ma?
are you chinese?



我 不 是 中国人，我 是 美国人
wǒ bú shì zhōng guó rén, wǒ shì měi guó rén.
im not Chinese, im American
